华为配置专题03DHCPVLAN配置

PC1和PC2跨VLAN通信实验，通过SW三层交换机

undo info-center enable 关闭设备自动消息提示

display vlan summary 查看现有vlan

一、
三层启VLAN接口

vlan 10

interface Vlanif 10 进入VLANIF10接口

ip address 192.168.10.254 24 配置IP地址为 192.168.10.254，掩码长度为24位

vlan 20

interface Vlanif 20 进入VLANIF20接口

ip address 192.168.20.254 24 配置IP地址为 192.168.20.254，掩码长度为24位

quit

二、
二层，把相应接口放到对应的VLAN里

创建了3层的vlan接口后

（VLANIF接口是基于VLAN的三层逻辑接口，可以配置IP地址。划分VLAN后，同一VLAN内的用户可以互相通信，但是属于不同VLAN的用户不能直接通信。为了实现VLAN间通信，可通过配置逻辑的三层接口——VLANIF接口来实现）

interface GigabitEthernet 0/0/1 进入千兆以太网接口 G 0/0/1

port link-type access 配置端口链路类型位ACCESS

port default vlan 10 配置端口的默认VLAN为10

interface GigabitEthernet 0/0/2 进入千兆以太网接口 G 0/0/2

port link-type access 配置端口链路类型位ACCESS

port default vlan 20 配置端口的默认VLAN为20

参考链接：
https://www.itmemo.cn/html/819.html

display ip routing-table

把三层交换机作为DHCP服务器

基于接口地址池的DHCP

dhcp enable 开启DHCP功能

interface Vlanif 10 进入接口Vlanif 10

dhcp select interface 选择基于接口地址池，把接口ip地址段作为DHCP地址池

dhcp server dns-list 8.8.8.8 指定接口地址池下的DNS服务器地址

dhcp server excluded-ip-address 192.168.10.101 192.168.10.253 配置接口地址池中不参与自动分配的ip地址范围

dhcp server lease day 8 配置DHCP服务器接口地址池中ip地址的租用有效期功能，缺省情况下，ip地址的租期为1天

display this

在PC1接口处抓包看还有 Gratuitous ARP（免费ARP）：
去请求自己ip地址的mac地址
发Gratuitous ARP做冲突检测
Gratuitous ARP 免费ARP，用来检测IP地址是否冲突

开始配置PC2

基于全局地址池的DHCP

interface Vlanif 20
dhcp select global
quit

ip pool 20 创建全局地址池

network 192.168.20.0 mask 24 配置全局地址池可动态分配的IP地址范围（宣告）

gateway-list 192.168.20.254 配置DHCP客户端的网关地址

dns-list 9.9.9.9 指定接口地址池下的DNS服务器地址

lease day 10 配置DHCP服务器接口地址池中ip地址的租用有效期功能，缺省情况下，ip地址的租期为1天

excluded-ip-address 192.168.20.151 192.168.20.253 配置接口地址池中不参与自动分配的ip地址范围

quit

display current-configuration 查看全局地址池配置

把PC1也变成基于全局地址池

interface Vlanif10

undo dhcp select interface

display this

dhcp select global

ip pool 10

network 192.168.10.0 mask 24 配置全局地址池可动态分配的IP地址范围（宣告）

dns-list 8.8.8.8

lease 8

流量上来时先会判断属于哪个VLAN，会跟VLANIF10相同网关地址，也就是pool 10地址

relay，DHCP服务器和DHCP客户端不在同一网段

扩展知识点

一、DHCP Relay 中继
如果LSW2是DHCP服务器，路由器可以隔离广播，PC1和PC2的广播就不能到LSW2了
通过DHCP Relay，DHCP广播转换成单播，发给DHCP服务器

二、DHCP Snooping

dhcp snooping enable

有其他DHCP服务器，插入家用路由器（默认是一个DHCP服务器）
启用snooping功能，把GE 0/0/3设置成trust接口，其他接口都是unTrust
其他任何地方来的DHCP地址都不接受

interface GigabitEthernet 0/0/1
DHCP snooping trusted

三、DHCP选项

要配置到pool里

interface VLANIF10

ip pool 20

option 43

跟下面的主机分配ip地址同时，再分配一个ip地址
无线里面经常用
DHCP
option 43
给AP分ip地址的同时，告诉AP AC的地址是多少，AP要找AC，通过AC对AP进行全局控制
